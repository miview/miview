# miview
> 基于element-ui二次开发的公共组件库

## package.json 说明文档

### scripts

- `bootstrap`: 安装依赖

- `dev`: 本地开发模式

- `dev:site`: 本地site站点生成预览

- `build:entry`: 自动生成 `packages/index.js`

- `build:site`: 线上site站点生成并部署
